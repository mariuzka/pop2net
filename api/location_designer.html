LocationDesigner

The LocationDesigner class is an interface to design specific network structures using the Creator.

class pop2net.LocationDesigner(*args, **kwargs)[source]

Helper class to create locations from inside the Creator.

bridge(actor)[source]

Create locations with one actor for each unique value returned.

Cannot be used in combination with melt().

Parameters:

actor (_actor.Actor) – The actor that is currently processed by the Creator.

Returns:

The value which is used to assign actors to location instances.

Return type:

float | str | list | None

filter(actor)[source]

Check whether the actor is meant to join this type of location.

Parameters:

actor (Actor) – The actor that is currently processed by the Creator.

Return type:

bool

Returns:

True if the actor is allowed to join the location, False otherwise. Returns True by default.

melt()[source]

Allows to assign actors with different attribute values to the same location.

Merges the actors assigned to the instances of the returned location classes into one instance of this location class.

Returns:

A list or tuple of location classes.

Return type:

None | list | tuple

mutate()[source]

dict

nest()[source]

Nests this location class into another location class.

Ensures that the actors assigned to the same instance of this location class are also assigned to the same instance of the returned location class.

Returns:

The location class in which this location class should be nested.

Return type:

type[Location] | None
