Inspecting the network: The Inspector classÔÉÅ

When building network models, it is important to evaluate the created networks to ensure they are a valid representation of the target network. For this purpose, pop2net provides a couple of tools.

[1]:

import pandas as pd

import pop2net as p2n

df_school = pd.read_csv("example_school_data.csv", sep=";")


class TeachersInClassRoom(p2n.MeltLocation):
    def setup(self):
        self.n_agents = 1

    def filter(self, agent):
        return agent.status == "teacher"


class PupilsInClassRoom(p2n.MeltLocation):
    def setup(self):
        self.n_agents = 4

    def filter(self, agent):
        return agent.status == "pupil"

    def split(self, agent):
        return agent.grade

    def stick_together(self, agent):
        return agent.friend_group


class ClassRoom(p2n.MagicLocation):
    def melt(self):
        return [TeachersInClassRoom, PupilsInClassRoom]

    def weight(self, agent):
        return agent.hours * 10

    def project_weights(self, agent1, agent2) -> float:
        return min([self.get_weight(agent1), self.get_weight(agent2)])


class School(p2n.MagicLocation):
    n_locations = 2

    def stick_together(self, agent):
        return agent.ClassRoom


class SoccerTeam(p2n.MagicLocation):
    n_agents = 11

    def nest(self):
        return School


model = p2n.Model()
creator = p2n.Creator(model)
inspector = p2n.NetworkInspector(model)

creator.create(
    df=df_school,
    location_classes=[
        ClassRoom,  # nested into School using `stick_together()`
        School,
        SoccerTeam,  # nested into School using `nest()`
    ],
)
inspector.plot_agent_network(agent_attrs=df_school.columns, agent_color="status")

Affiliation statisticsÔÉÅ

To get a quick overview of the number of affiliations per agents or location, use inspector.eval_affiliations(). The first table shows how many agents are assigned to an instance of a location. The second table shows the number of locations an agent is assigned to.

Contact matrixÔÉÅ

inspector.create_contact_matrix() shows the contact intensity between agents using a contact matrix. With no further input than the list of agents, the matrix shows the contact intensities for each pair of agents.
